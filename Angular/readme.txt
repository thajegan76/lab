Install Angular CLI 
Open the command prompt and run the npm install @angular/cli --g 
ng version

Creating Angular Project
Navigate to a specific folder where you want to create the angular project
ng new ecommerce

ng serve
1) compile all the typescript code to javascript
2) bundle the javascript codes to a single file
3) add the file into index.html
4) deploy into light weight webserver
5) start the server at port 4200
